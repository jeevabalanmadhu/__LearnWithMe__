Python Lists
------------

	1.List is a collection which is ordered and changeable or Mutable. 

	2.Allows duplicate members.

	3.You access the list items by referring to the index number


Example
--------
Create a List:

MyList = [10,3.14,"apple",4+5j,True]
print(MyList)
-------------------------------------------------------------------------------------------------------------------------------------
Access Items
------------

	1.You access the list items by referring to the index number:

Example
-------
MyList = [10,3.14,"apple",4+5j,True]

#Using Positive Index
print(MyList[0])
print(MyList[1])
print(MyList[2])
print(MyList[3])
print(MyList[4])


#Using Negative Index
print(MyList[-5])
print(MyList[-4])
print(MyList[-3])
print(MyList[-2])
print(MyList[-1])
-------------------------------------------------------------------------------------------------------------------------------------
Change Item Value
-----------------
	
	1.To change the value of a specific item, refer to the index number:

Example
-------
MyList = [10,3.14,"apple",4+5j,True]

MyList[2]="banana"

print(MyList)
--------------------------------------------------------------------------------------------------------------------------------------
Loop Through a List
-------------------

	1.You can loop through the list items by using a for loop:

Example
--------
#Print all items in the list, one by one:

MyList = [10,3.14,"apple",4+5j,True]

for i in MyList:
  print(i)
--------------------------------------------------------------------------------------------------------------------------------------
Check if Item Exists
--------------------

	1.To determine if a specified item is present in a list use the in keyword:

Example
-------
#Check if "apple" is present in the list:

MyList = [10,3.14,"apple",4+5j,True]

if "apple" in MyList:
  print("Yes, 'apple' is in the list")
-------------------------------------------------------------------------------------------------------------------------------------
List Length
-----------

	1.To determine how many items a list has, use the len() method:

Example
-------
#Print the number of items in the list:

MyList = [10,3.14,"apple",4+5j,True]
print(len(MyList))
-------------------------------------------------------------------------------------------------------------------------------------
Delete List
-----------

	1.To delete the list completely

Example
-------

#The del keyword can also delete the list completely:

MyList = [10,3.14,"apple",4+5j,True]
del MyList
-------------------------------------------------------------------------------------------------------------------------------------
The list() Constructor
----------------------

	1.It is also possible to use the list() constructor to make a list.

Example
-------
#Using the list() constructor to make a List:

MyList = list(("apple", "banana", "cherry")) # note the double round-brackets
print(MyList)
--------------------------------------------------------------------------------------------------------------------------------------
List Methods
------------

Python has a set of built-in methods that you can use on lists.

Method				Description
--------------------------------------------------------------------------------------------------------------------
append()			Adds an element at the end of the list

clear()				Removes all the elements from the list

copy()				Returns a copy of the list

count()				Returns the number of elements with the specified value

extend()			Add the elements of a list (or any iterable), to the end of the current list

index()				Returns the index of the first element with the specified value

insert()			Adds an element at the specified position

pop()				Removes the element at the specified position

remove()			Removes the item with the specified value

reverse()			Reverses the order of the list

sort()				Sorts the list
